StatTRACK — Setup Instructions
================================

IMPORTANT — DO THIS FIRST:
  Move this entire folder to your Desktop or Documents before opening anything.
  (Do not run it directly from Downloads or a zip file.)

----------------------------------------

MAC (first time only):
  1. Move this folder to Desktop or Documents
  2. Right-click StatTRACK.app → Open
  3. Click "Open" when macOS warns about unidentified developer
  4. Click "Install" if asked to set up Python tools (~30 seconds, once only)
  5. The first launch also downloads Python 3.12 — this takes 1-2 minutes once
  6. Your browser opens to http://localhost:5001

  After first setup, just double-click StatTRACK.app to launch instantly.

WINDOWS (first time only):
  1. Move this folder to Desktop or Documents
  2. Double-click launch_windows.bat
  3. Click "More info" → "Run anyway" if Windows warns you
  4. Wait ~1 minute for Python tools to install (once only)
  5. Your browser opens to http://localhost:5001

  After first setup, just double-click launch_windows.bat to launch.

----------------------------------------

To stop the app: close the Terminal / command prompt window that opened.
Your data is saved in mediavault.db inside this folder.
